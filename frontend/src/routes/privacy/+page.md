# Política de Privacidade e Proteção de Dados

## Sobre os Dados dos Contratos

### Origem dos Dados

Todos os dados de contratos públicos disponibilizados no **Contrato Público** são recolhidos exclusivamente do [Portal BASE](https://www.base.gov.pt/base4) (Base de Dados de Contratação Pública), que é o portal oficial do Instituto dos Mercados Públicos, do Imobiliário e da Construção (IMPIC).

O **Contrato Público** atua como um agregador e motor de pesquisa, facilitando o acesso e pesquisa destes dados públicos através de uma interface mais rápida, intuitiva e funcional. Não somos afiliados ao IMPIC nem ao Governo de Portugal.

### Apresentação dos Dados

Os dados são apresentados tal como se encontram disponibilizados no Portal BASE. O **Contrato Público**:

- **Não modifica** o conteúdo dos dados dos contratos
- **Não valida** a exatidão ou atualidade das informações
- **Não adiciona** informações que não estejam disponíveis no Portal BASE
- **Apenas agrega e indexa** os dados para proporcionar uma melhor experiência de pesquisa

### Correções e Atualizações

Para qualquer correção, atualização ou remoção de dados relativos a contratos públicos, os pedidos devem ser direcionados diretamente ao **IMPIC** através do Portal BASE, uma vez que somos apenas um agregador de informação pública.

As alterações efetuadas no Portal BASE serão refletidas no **Contrato Público** no próximo ciclo de atualização de dados, que ocorre periodicamente.

## Responsabilidade e Limitações

O **Contrato Público** esforça-se por manter os dados sincronizados com o Portal BASE, mas:

- Não garantimos a atualidade em tempo real dos dados
- Não nos responsabilizamos por erros ou imprecisões nos dados originais do Portal BASE
- Não nos responsabilizamos por atrasos na atualização dos dados
- Recomendamos sempre a consulta do Portal BASE para confirmação oficial de informações críticas

## Proteção de Dados Pessoais (RGPD)

### Dados dos Contratos

Os dados dos contratos públicos disponibilizados no **Contrato Público** são considerados informação pública nos termos da legislação portuguesa e europeia. Estes dados incluem informações sobre entidades contratantes e contratadas (nomes, NIFs, etc.) que fazem parte dos registos oficiais de contratação pública.

A disponibilização destes dados está em conformidade com:
- **Regulamento Geral sobre a Proteção de Dados (RGPD)**
- **Lei de Acesso aos Documentos Administrativos**
- **Legislação sobre Contratação Pública**

### Dados dos Utilizadores da Plataforma

O **Contrato Público** respeita a privacidade dos seus utilizadores:

- **Não recolhemos** dados pessoais dos utilizadores
- **Não utilizamos** sistemas de autenticação ou registo
- **Não armazenamos** histórico de pesquisas dos utilizadores
- **Não utilizamos** cookies de tracking ou análise comportamental
- **Não partilhamos** quaisquer dados com terceiros

### Dados Técnicos

A plataforma pode recolher dados técnicos anónimos e agregados para fins de:
- Manutenção e otimização do serviço
- Estatísticas de utilização gerais
- Identificação e resolução de problemas técnicos

Estes dados não permitem a identificação individual dos utilizadores.

## Segurança dos Dados

Implementamos medidas técnicas e organizacionais adequadas para proteger os dados:

- Conexões seguras (HTTPS)
- Proteção contra acessos não autorizados
- Backups regulares dos dados públicos
- Monitorização de segurança

## Alterações a Esta Política

Esta política de privacidade pode ser atualizada periodicamente para refletir alterações no funcionamento da plataforma ou na legislação aplicável. A versão mais recente estará sempre disponível nesta página.
